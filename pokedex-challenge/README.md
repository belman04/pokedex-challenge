# Pokédex Challenge

Este proyecto es una Pokédex básica hecha con React. Permite ver Pokémon, buscarlos por nombre, filtrarlos por tipo y ordenarlos de forma sencilla.

## Funcionalidades

- Mostrar una lista de Pokémon
- Buscar Pokémon por nombre
- Filtrar por tipo
- Ordenar alfabéticamente
- Mostrar datos básicos como experiencia base, peso, altura y tipo

## Tecnologías utilizadas

- React (Framework principal)
- Axios (Cliente HTTP para el consumo de datos desde la API)
- PokeAPI (Fuente de datos oficial de los Pokémon)
- Lucide React (Librería de iconos)
